🎯 FUNCIONALIDADES DETALHADAS
═══════════════════════════════════════════════════════════════════════════════

CANETA ✏️
  • Desenho livre e natural
  • Traço suave (lineCap: round)
  • Recomendado: 2-5px

BORRACHA 🧹
  • Remove pixels desenhados
  • Funciona bem para corrigir
  • Recomendado: 10-20px

LINHA 📏
  • Linha reta entre dois pontos
  • Clique para início, clique para fim
  • Recomendado: 2-3px

RETÂNGULO ◻️
  • Quadrado ou retângulo
  • Somente contorno (stroke)
  • Tamanho: Clique e arraste
  • Recomendado: 2-4px

CÍRCULO ⭕
  • Círculo perfeito
  • Somente contorno (stroke)
  • Raio: Distância do clique até soltura
  • Recomendado: 2-4px

📊 TÉCNICAS
═══════════════════════════════════════════════════════════════════════════════

Canvas 2D Context:
  • beginPath() / stroke()
  • lineTo() / moveTo()
  • strokeRect() / fillRect()
  • arc() para círculos
  • clearRect() para apagar

Performance:
  • Renderização nativa do navegador
  • Muito rápida (60 FPS)
  • Sem lag mesmo com traços grandes

Auto-save:
  • canvas.toDataURL('image/png')
  • Salvo em localStorage
  • Restaurado ao carregar página
